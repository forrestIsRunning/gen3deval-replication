# 前端使用

## 启动

```bash
cd /Users/xiaoxia/Projects/experiments/gen3deval-replication
uv sync
cp .env.example .env
uv run uvicorn web.app:app --reload --port 7860
```

浏览器打开：

```text
http://127.0.0.1:7860
```

## 使用流程

1. 先准备 manifest：

```bash
uv run python scripts/prepare_objaverse_sample.py --num-assets 120
uv run python scripts/download_objaverse_assets.py --manifest data/processed/manifest_120.jsonl
```

2. 安装 Blender 并渲染：

```bash
blender -b --python scripts/render_blender.py -- \
  --manifest data/processed/manifest_120.jsonl \
  --output-dir data/renders \
  --views 4
```

3. 在前端选择模型、manifest 和 limit，点击 Run。

4. 页面会显示：

- 3D 模型预览；
- 每个资产的多维评分；
- 数据集平均分柱状图；
- VLM 给出的理由。

当前前端已经去掉 mock 入口。`Run VLM` 只会调用真实 LiteLLM/VLM 服务；如果 key 无权限或服务返回 403，页面不会伪造评分。几何指标、渲染成功率和论文覆盖仍然来自本地真实实验产物。

可选 VLM 只保留：

- `qwen3-vl-plus`
- `qwen3-vl-flash`

`Run VLM` 是后台任务，点击后会立刻返回 job id 并轮询状态；如果没有 `.env` 中的 `LITELLM_API_KEY`，日志会显示 `Missing LITELLM_API_KEY in .env`。

## 没有 Blender 时

没有 Blender 时不要运行 VLM 评分，因为 VLM 评分依赖真实 RGB/Normal 渲染图。可以只查看已有的几何指标、论文覆盖和已经生成的渲染实验结果。
